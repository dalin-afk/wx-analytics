# 公众号数据解析系统

一个基于 Streamlit 的 Web 应用，用于解析微信公众平台后台导出的 Excel 数据文件。

## 功能特点

- 📂 支持批量上传 `.xls` / `.xlsx` 文件
- 📊 自动提取 17 项核心指标（阅读人数、送达人数、各渠道分布、点赞、分享等）
- 📈 可视化展示各渠道阅读人数分布
- 💾 支持导出 Excel / CSV 汇总表
- 🔒 数据仅在浏览器本地处理，不上传至任何服务器

## 本地运行

```bash
pip install -r requirements.txt
streamlit run app.py
```

## 免费部署到 Streamlit Cloud

1. 将此项目推送到 GitHub 公开仓库
2. 访问 [share.streamlit.io](https://share.streamlit.io) 登录
3. 点击 "New app" → 选择 GitHub 仓库 → 选择 `app.py`
4. 点击 Deploy，几分钟后获得永久免费链接

## 支持的数据字段

| 字段 | 说明 |
|------|------|
| 文章标题 | 从 Excel B1/C1 合并单元格提取 |
| 全部阅读人数 | 数据概况区域 |
| 送达人数 | 阅读转化区域 |
| 推荐/其他/消息/主页/搜一搜/朋友圈/聊天 阅读人数 | 数据趋势明细区域 |
| 阅读后关注人数 | 数据概况区域 |
| 点赞人数 | 数据概况区域 |
| 评论次数 | 数据概况区域 |
| 总分享人数 | 数据概况区域 |
| 在看人数 | 数据概况区域 |
| 停留时间 | 数据概况区域（秒） |
| 完读率 | 数据概况区域 |
